# AI 工程师课程

## 课程简介
AI 工程师培养课程，从 Python 基础到深度学习，覆盖 NLP、计算机视觉、大模型应用开发，培养企业级 AI 开发能力。

## 适合人群
- 有一定编程基础的开发者（不限语言）
- 数学/统计背景想转 AI 的学生
- 传统软件开发想转型 AI 的工程师
- 数据分析师想提升到机器学习方向

## 前置要求
- 至少掌握一门编程语言（Python 优先）
- 高中数学基础（函数、概率、线性代数基本概念）
- 非必需但有帮助：了解 Linux 基本操作

## 课程大纲
1. Python 数据分析基础（NumPy、Pandas、Matplotlib）
2. 机器学习原理（监督学习、无监督学习、模型评估）
3. 深度学习基础（神经网络、CNN、RNN、Transformer）
4. 自然语言处理（文本分类、命名实体识别、文本生成）
5. 计算机视觉（图像分类、目标检测、图像分割）
6. 大模型应用开发（LangChain、RAG、Agent、Prompt Engineering）
7. MLOps 基础（模型部署、Docker、API 服务化）
8. 综合项目（智能客服系统 / 图像识别平台 / RAG 知识库问答）
