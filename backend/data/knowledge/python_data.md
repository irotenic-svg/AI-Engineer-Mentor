# Python 数据分析课程

## 课程简介
Python 数据分析课程面向零基础学员，从 Python 基础语法开始，逐步深入数据处理、可视化和机器学习入门。

## 适合人群
- 零基础编程新手
- 想转行数据分析的在职人员（财务、运营、市场等）
- 对 Python 和数据科学感兴趣的学生
- 日常工作中需要处理数据的职场人

## 前置要求
- 基础计算机操作能力
- 高中数学水平（基础统计知识会随堂补充）
- 无需编程经验

## 课程大纲
1. Python 基础语法（变量、数据类型、控制流、函数）
2. NumPy 数值计算（数组操作、矩阵运算、广播机制）
3. Pandas 数据处理（数据清洗、分组聚合、时间序列、合并连接）
4. Matplotlib + Seaborn 数据可视化（折线图、柱状图、散点图、热力图、仪表板布局）
5. Scikit-learn 机器学习入门（回归、分类、聚类、模型评估）
6. SQL 基础与数据库查询（MySQL、数据提取与报表）
7. Excel 与 Python 联动（openpyxl、自动化报表生成）
8. 综合实战项目（电商用户行为分析 / 销售数据预测 / 可视化仪表板）

## 常见问题
Q: 零基础能学会吗？
A: 完全可以。课程从最基础的 Python 语法开始，循序渐进，配合大量实战练习。全日制前 2 周为基础适应期。

Q: 需要数学基础吗？
A: 基础数学即可（高中水平）。课程中涉及的统计知识会随堂补充，不需提前准备。

Q: 数据分析需要的工具课程都教吗？
A: 课程覆盖 NumPy、Pandas、Matplotlib、Seaborn、Scikit-learn、SQL、Excel 等核心工具。

Q: 结课后有就业支持吗？

Q: 课程和网上免费资源有什么不同？
