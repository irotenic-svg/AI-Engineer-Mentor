# Java 全栈开发课程

## 课程简介
Java 全栈开发课程，从 Java 基础到 Spring Boot 微服务，涵盖数据库、前端基础、项目部署，培养企业级后端开发能力。

## 适合人群
- 零基础想学后端开发的转行者
- 计算机专业在校学生
- 其他语言开发者想转 Java 生态
- 初级 Java 开发想系统提升

## 课程大纲
1. Java 基础（语法、OOP、集合框架、IO）
2. 数据库基础（MySQL、Redis、SQL 优化）
3. Java Web（Servlet、JSP、Tomcat）
4. Spring 框架（IoC、AOP、Spring MVC）
5. Spring Boot 微服务（RESTful API、MyBatis-Plus、JPA）
6. Spring Cloud 分布式（Nacos、Gateway、Feign、Sentinel）
7. 前端基础（HTML/CSS/JS + Vue 入门，够用即可）
8. 项目实战（企业 OA 系统、电商平台）

## 常见问题
Q: Java 现在还值得学吗？
A: Java 仍然是企业级后端的主流语言，金融、电商、政务等领域需求旺盛，岗位基数大。

Q: 课程包含分布式和微服务吗？
A: 包含。Spring Cloud 模块完整覆盖 Nacos、Gateway、Feign、Sentinel 等技术栈。

Q: 学完 Java 和学完 Python 的就业方向有什么不同？
A: Java 主要走后端/企业级系统方向，去向偏金融、电商、政企；Python 偏数据分析、AI、自动化方向。

Q: 课程项目是真实项目吗？
A: 项目来源于讲师过往的企业级项目简化版，涵盖完整的需求分析→设计→开发→部署流程。
