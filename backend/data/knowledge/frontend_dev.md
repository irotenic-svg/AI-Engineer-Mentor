# 前端开发工程师课程

## 课程简介
前端开发工程师课程面向零基础及初级开发者，系统学习 HTML/CSS/JavaScript 三大基础，深入 React/Vue 主流框架，覆盖工程化、性能优化与项目实战。

## 适合人群
- 零基础想进入 IT 行业的转行者
- 有一定自学基础但缺乏系统训练的初学者
- UI/UX 设计师想拓展前端技能
- 后端开发想转全栈的工程师

## 课程大纲
1. HTML5 & CSS3 基础（语义化标签、Flexbox、Grid 布局）
2. JavaScript 核心（ES6+、异步编程、DOM 操作）
3. React 框架（组件化、Hooks、状态管理 Redux/Zustand）
4. Vue 框架（Composition API、Pinia、Vue Router）
5. TypeScript 入门与实践
6. 前端工程化（Vite、Webpack、ESLint、Prettier）
7. Node.js 基础与全栈入门
8. 综合实战项目（电商前端、后台管理系统）

## 常见问题
Q: 完全零基础能学会吗？
A: 可以。课程从 HTML 标签开始教，前 2 周为基础适应期，配有助教一对一辅导。

Q: 需要自备什么电脑？
A: Windows/Mac 均可，8G 内存以上，课程开始前会指导安装开发环境。

Q: 全日制和周末班课程内容一样吗？
A: 完全一样，只是节奏不同。全日制每周 5 天全天上课，周末班仅周六日。

Q: 结课后能马上找工作吗？
A: 课程最后 2 周为就业冲刺阶段，包括模拟面试和简历优化，往期学员平均 4 周内拿到 offer。
